## Principais Componentes Bootstrap Utilizados no Frontend

Este documento descreve os principais componentes do Bootstrap 5.3.3 utilizados na aplicação "workOut-app", explicando sua finalidade e aplicação.

### 1. Navbar (Barra de Navegação)
*   **Finalidade:** Fornece uma barra de navegação responsiva e fixa no topo da página, permitindo a navegação entre as diferentes seções da aplicação.
*   **Classes Bootstrap Principais:**
    *   [`navbar`](https://getbootstrap.com/docs/5.3/components/navbar/#how-it-works): O contêiner principal da barra de navegação.
    *   [`navbar-expand-lg`](https://getbootstrap.com/docs/5.3/components/navbar/#responsive-behaviors): Define o ponto de interrupção (breakpoint) a partir do qual a barra de navegação se expande (neste caso, para telas grandes e maiores).
    *   [`fixed-top`](https://getbootstrap.com/docs/5.3/components/navbar/#placement): Fixa a barra de navegação no topo da viewport.
    *   [`container-fluid`](https://getbootstrap.com/docs/5.3/layout/containers/): Um contêiner de largura total para o conteúdo da barra de navegação.
    *   [`navbar-brand`](https://getbootstrap.com/docs/5.3/components/navbar/#brand): Usado para o logotipo ou nome da aplicação.
    *   [`navbar-toggler`](https://getbootstrap.com/docs/5.3/components/navbar/#toggler): O botão que alterna a navegação em telas menores.
    *   [`navbar-toggler-icon`](https://getbootstrap.com/docs/5.3/components/navbar/#toggler-icon): O ícone do botão de alternância.
    *   [`collapse`](https://getbootstrap.com/docs/5.3/components/collapse/): Usado em conjunto com `navbar-collapse` para o comportamento de recolhimento da navegação.
    *   [`navbar-collapse`](https://getbootstrap.com/docs/5.3/components/navbar/#supported-content): O contêiner para os itens de navegação que podem ser recolhidos.
    *   [`navbar-nav`](https://getbootstrap.com/docs/5.3/components/navbar/#nav): Uma lista de itens de navegação.
    *   [`nav-item`](https://getbootstrap.com/docs/5.3/components/navbar/#nav): Um item individual na lista de navegação.
    *   [`nav-link`](https://getbootstrap.com/docs/5.3/components/navbar/#nav): O link dentro de um item de navegação.
*   **Aplicação:** A barra de navegação é o ponto de entrada principal para o usuário interagir com as diferentes seções do painel de controle (Dashboard, Clientes, Planos de Treino, Aulas). O design responsivo garante que ela se adapte bem a dispositivos móveis e desktops.

### 2. Cards
*   **Finalidade:** Usados para exibir conteúdo de forma organizada e visualmente distinta, como os cartões de estatísticas no Dashboard.
*   **Classes Bootstrap Principais:**
    *   [`card`](https://getbootstrap.com/docs/5.3/components/card/): O contêiner principal do cartão.
    *   [`card-body`](https://getbootstrap.com/docs/5.3/components/card/#body): O corpo do cartão, onde o conteúdo principal é colocado.
    *   [`card-title`](https://getbootstrap.com/docs/5.3/components/card/#titles-text-and-links): Título do cartão.
*   **Aplicação:** No Dashboard, os cards são usados para apresentar métricas importantes como "Clientes Ativos", "Planos Cadastrados", "Próximas Aulas" e "Novas Notificações", cada um com um ícone e um número de destaque. Na seção de Clientes, um card maior envolve a tabela de clientes e o formulário de busca.

### 3. Grid System (Sistema de Colunas)
*   **Finalidade:** Permite criar layouts responsivos e flexíveis, dividindo o espaço horizontal em colunas.
*   **Classes Bootstrap Principais:**
    *   [`row`](https://getbootstrap.com/docs/5.3/layout/grid/#rows): Um contêiner para colunas.
    *   [`col-xl-*`](https://getbootstrap.com/docs/5.3/layout/grid/#grid-options), [`col-md-*`](https://getbootstrap.com/docs/5.3/layout/grid/#grid-options), [`col-12`](https://getbootstrap.com/docs/5.3/layout/grid/#grid-options): Classes que definem a largura das colunas em diferentes tamanhos de tela (extra-grandes, médias, e para todas as telas, respectivamente).
    *   [`g-4`](https://getbootstrap.com/docs/5.3/layout/gutters/): Adiciona espaçamento (gutter) entre as colunas.
*   **Aplicação:** Utilizado extensivamente no Dashboard para organizar os cartões de estatísticas em um layout de grade responsivo. Também usado nos modais para organizar os campos do formulário.

### 4. Formulários (Forms)
*   **Finalidade:** Cria elementos de entrada de dados estilizados e responsivos.
*   **Classes Bootstrap Principais:**
    *   [`form-label`](https://getbootstrap.com/docs/5.3/forms/overview/#labels): Rótulo para campos de formulário.
    *   [`form-control`](https://getbootstrap.com/docs/5.3/forms/form-control/): Estiliza inputs de texto, email, telefone, etc.
    *   [`form-select`](https://getbootstrap.com/docs/5.3/forms/select/): Estiliza elementos `<select>`.
    *   [`row`](https://getbootstrap.com/docs/5.3/layout/grid/#rows), [`g-3`](https://getbootstrap.com/docs/5.3/layout/gutters/): Usados para organizar campos de formulário em um layout de grade.
*   **Aplicação:** Presente no modal de "Cadastrar Novo Cliente" para coletar informações do cliente e na barra de busca da tabela de clientes.

### 5. Botões (Buttons)
*   **Finalidade:** Elementos interativos para ações do usuário.
*   **Classes Bootstrap Principais:**
    *   [`btn`](https://getbootstrap.com/docs/5.3/components/buttons/): A classe base para botões.
    *   [`btn-primary`](https://getbootstrap.com/docs/5.3/components/buttons/#examples): Botão com a cor primária do tema.
    *   [`btn-secondary`](https://getbootstrap.com/docs/5.3/components/buttons/#examples): Botão com a cor secundária do tema.
    *   [`btn-sm`](https://getbootstrap.com/docs/5.3/components/buttons/#sizes): Botão pequeno.
    *   [`btn-link`](https://getbootstrap.com/docs/5.3/components/buttons/#link): Botão que se parece com um link.
    *   [`rounded-pill`](https://getbootstrap.com/docs/5.3/utilities/borders/#border-radius): Adiciona bordas arredondadas completas.
*   **Aplicação:** Usados para "Novo Cliente", "Salvar Cliente", "Cancelar", "Confirmar Exclusão" e as ações de "Ver", "Editar" e "Excluir" na tabela de clientes. O botão de alternância de tema também utiliza classes de botão.

### 6. Modals
*   **Finalidade:** Caixas de diálogo flutuantes que exigem interação do usuário, como formulários ou confirmações.
*   **Classes Bootstrap Principais:**
    *   [`modal`](https://getbootstrap.com/docs/5.3/components/modal/): O contêiner principal do modal.
    *   [`fade`](https://getbootstrap.com/docs/5.3/components/modal/#how-it-works): Adiciona uma transição de fade ao modal.
    *   [`modal-dialog`](https://getbootstrap.com/docs/5.3/components/modal/#sizing): O contêiner para o conteúdo do modal.
    *   [`modal-lg`](https://getbootstrap.com/docs/5.3/components/modal/#sizing): Define um modal grande.
    *   [`modal-dialog-centered`](https://getbootstrap.com/docs/5.3/components/modal/#vertically-centered): Centraliza o modal verticalmente.
    *   [`modal-content`](https://getbootstrap.com/docs/5.3/components/modal/#content): O conteúdo real do modal.
    *   [`modal-header`](https://getbootstrap.com/docs/5.3/components/modal/#header): O cabeçalho do modal.
    *   [`modal-title`](https://getbootstrap.com/docs/5.3/components/modal/#header): O título do modal.
    *   [`btn-close`](https://getbootstrap.com/docs/5.3/components/close-button/): O botão de fechar (X) no cabeçalho do modal.
    *   [`modal-body`](https://getbootstrap.com/docs/5.3/components/modal/#body): O corpo do modal, onde o conteúdo principal é colocado.
    *   [`modal-footer`](https://getbootstrap.com/docs/5.3/components/modal/#footer): O rodapé do modal, geralmente para botões de ação.
*   **Aplicação:** Utilizado para o formulário de "Cadastrar Novo Cliente" e para a caixa de diálogo de "Confirmar Exclusão".

### 7. Tabelas (Tables)
*   **Finalidade:** Exibir dados tabulares de forma clara e organizada.
*   **Classes Bootstrap Principais:**
    *   [`table`](https://getbootstrap.com/docs/5.3/content/tables/): A classe base para tabelas.
    *   [`table-hover`](https://getbootstrap.com/docs/5.3/content/tables/#hoverable-rows): Adiciona um estado de hover às linhas da tabela.
    *   [`align-middle`](https://getbootstrap.com/docs/5.3/utilities/vertical-align/): Alinha o conteúdo das células verticalmente ao meio.
    *   [`table-responsive`](https://getbootstrap.com/docs/5.3/content/tables/#responsive-tables): Torna a tabela responsiva, adicionando barras de rolagem horizontais em telas menores.
*   **Aplicação:** Usado para exibir a lista de clientes na seção "Gerenciar Clientes".

### 8. Badges
*   **Finalidade:** Pequenos componentes de contagem e rotulagem.
*   **Classes Bootstrap Principais:**
    *   [`badge`](https://getbootstrap.com/docs/5.3/components/badge/): A classe base para badges.
    *   [`bg-success`](https://getbootstrap.com/docs/5.3/utilities/colors/#background-color), [`bg-warning`](https://getbootstrap.com/docs/5.3/utilities/colors/#background-color): Classes para definir a cor de fundo do badge.
*   **Aplicação:** Usado para indicar o "Status do Plano" dos clientes na tabela, com cores diferentes para "Ativo" e "Inativo".

### 9. Ícones Bootstrap (Bootstrap Icons)
*   **Finalidade:** Uma biblioteca de ícones SVG que podem ser facilmente incorporados em componentes Bootstrap.
*   **Classes Bootstrap Principais:**
    *   [`bi`](https://icons.getbootstrap.com/): Classe base para todos os ícones Bootstrap.
    *   [`bi-activity`](https://icons.getbootstrap.com/icons/activity/): Ícone de atividade (usado no navbar-brand).
    *   [`bi-moon-stars-fill`](https://icons.getbootstrap.com/icons/moon-stars-fill/), [`bi-sun-fill`](https://icons.getbootstrap.com/icons/sun-fill/): Ícones para o alternador de tema.
    *   [`bi-people-fill`](https://icons.getbootstrap.com/icons/people-fill/): Ícone de pessoas (usado no card de Clientes Ativos).
    *   [`bi-clipboard-data-fill`](https://icons.getbootstrap.com/icons/clipboard-data-fill/): Ícone de dados (usado no card de Planos Cadastrados).
    *   [`bi-calendar-check-fill`](https://icons.getbootstrap.com/icons/calendar-check-fill/): Ícone de calendário (usado no card de Próximas Aulas).
    *   [`bi-bell-fill`](https://icons.getbootstrap.com/icons/bell-fill/): Ícone de sino (usado no card de Novas Notificações).
    *   [`bi-plus-circle`](https://icons.getbootstrap.com/icons/plus-circle/): Ícone de adição (usado no botão "Novo Cliente").
    *   [`bi-eye-fill`](https://icons.getbootstrap.com/icons/eye-fill/): Ícone de olho (usado para "Ver" cliente).
    *   [`bi-pencil-fill`](https://icons.getbootstrap.com/icons/pencil-fill/): Ícone de lápis (usado para "Editar" cliente).
    *   [`bi-trash-fill`](https://icons.getbootstrap.com/icons/trash-fill/): Ícone de lixeira (usado para "Excluir" cliente).
*   **Aplicação:** Utilizados para adicionar elementos visuais e melhorar a usabilidade em botões, títulos de cards e no branding da aplicação.

### 10. Utilitários (Utilities)
*   **Finalidade:** Classes de uso único para estilização rápida e responsiva.
*   **Classes Bootstrap Principais:**
    *   [`d-flex`](https://getbootstrap.com/docs/5.3/utilities/flex/#enable-flex-behaviors): Habilita um contêiner flex.
    *   [`justify-content-between`](https://getbootstrap.com/docs/5.3/utilities/flex/#justify-content): Alinha itens flex ao longo do eixo principal.
    *   [`align-items-center`](https://getbootstrap.com/docs/5.3/utilities/flex/#align-items): Alinha itens flex ao longo do eixo transversal.
    *   [`mb-*`](https://getbootstrap.com/docs/5.3/utilities/spacing/#margin-and-padding), [`py-*`](https://getbootstrap.com/docs/5.3/utilities/spacing/#margin-and-padding), [`ms-*`](https://getbootstrap.com/docs/5.3/utilities/spacing/#margin-and-padding): Classes para margens e preenchimentos (margin-bottom, padding-vertical, margin-start).
    *   [`text-center`](https://getbootstrap.com/docs/5.3/utilities/text/#text-alignment): Centraliza o texto.
    *   [`text-decoration-none`](https://getbootstrap.com/docs/5.3/utilities/text/#text-decoration): Remove a decoração de texto (sublinhado).
    *   [`fw-bold`](https://getbootstrap.com/docs/5.3/utilities/text/#font-weight-and-italics): Define o peso da fonte como negrito.
    *   [`fs-4`](https://getbootstrap.com/docs/5.3/utilities/text/#font-sizes), [`fs-5`](https://getbootstrap.com/docs/5.3/utilities/text/#font-sizes): Define o tamanho da fonte.
*   **Aplicação:** Usados em toda a aplicação para ajustar espaçamentos, alinhamentos, tamanhos de fonte e outros detalhes de layout e tipografia.
